🧩 Shared Components

📦 Component Reference

Component ใช้ใน รายละเอียด
Profile Bar Step 2, 3, 4 แสดง avatar + ชื่อ + handle + ปุ่ม logout ครบทุก step
Wizard Steps donate.html 4 วงกลม + progress line แสดง step ปัจจุบัน/ผ่านแล้ว
Feeder Game Step 2 🥄 + 🐻 animation + ปุ่มตัก + bowl progress bar (0–1000g)
Suggested Amount Box Step 3 แสดงเมื่อ Step 2 มียอด — "ยอดโดเนทของคุณ ฿XXX"
File Drop Zone Step 4 click-to-upload · แสดง preview + filename + ขนาด
Slip Result Box Step 4 แสดงผล OCR: ยอด / ผู้รับ / บัญชี / ธนาคาร / เลขอ้างอิง / เวลา
Copy Button Step 3 คัดลอกเลขบัญชีไป clipboard พร้อม toast notification
Toast ทุกหน้า แจ้งเตือนชั่วคราว 2 วินาที (bottom center)
Wall of Love index.html รายการผู้โดเนท paginated (5/หน้า) เรียงล่าสุดก่อน
Bear-feeding Anim thank-you.html Avatar → bowl + bear + food bits flying (CSS animation)

🔌 API Endpoints

📡 Backend calls

EndpointMethodใช้ในหมายเหตุ
/getDonationAll GET index.html ยอดรวม + รายการทั้งหมดสำหรับ Wall of Love
/getUserInfoByXid GET Step 1 เช็ค user เดิม → ดึง user_id + ยอดสะสม
/saveUser POST Step 1 สร้าง user ใหม่ถ้ายังไม่มี
/saveDonate POST Step 4 บันทึกโดเนท (user_id, amount, slip_image, ref_code, …)
n8n webhook (OCR) POST Step 4 ส่ง form-data image → รับ slip data (amount, bank, ref, …)
Google Apps Script POST Step 4 อัปโหลดสลิปไป Google Drive (base64) — background call
